# Aniyomi Stream App

An independent streaming app featuring Aniyomi-like custom sources, including a built-in Ad & Popup Blocker, AnimePahe integration, and universal web streaming with redirect blocking.

## Aniyomi Extension JSON Link

If you are looking to add this to your Aniyomi app as an external plugin or extension repository, use the following JSON link:

```
https://raw.githubusercontent.com/<YOUR_GITHUB_USERNAME>/<YOUR_REPOSITORY_NAME>/main/index.min.json
```

### How to add to Aniyomi:
1. Open the Aniyomi app.
2. Go to **Settings > Browse > Extensions**.
3. Tap on **Extension repos > Add**.
4. Paste the JSON link above.
